# app-pieza-publicitaria
Crea una composición a partir de un producto, un slogan, un logotipo y un escenario con IA
El formato de la composición se puede elegir entre cuadrado, vertical u horizontal
Producto, slogan y logotipo se pueden desplazar y escalar sobre el escenario segun necesidades.
Se puede poner sombra al producto y cambiar su  intensidad y el desplazamiento horizontal y vertical para que se acomode a la luz del escenario.
Dispone de una API backend en un servidor de Red Descartes
